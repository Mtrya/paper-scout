# drafts/

Freeform scratch space for a run: the candidate-pool dump, working Lark DocxXML, skeleton and section fragments, temporary media dumps, and other intermediate files.

Overwrite freely; nothing here is durable or tracked. The final delivered DocxXML source belongs in `runs/<run-id>/report.docxxml`, and durable evidence belongs in the same run packet. Contents are gitignored; this README is tracked to keep the directory present on clone.
