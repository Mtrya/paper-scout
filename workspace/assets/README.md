# assets/

Extracted media for a run: figures, tables, and images pulled out of papers — anything binary or non-prose a deep dive or the report needs to point at.

Overwrite freely; nothing here is durable or tracked, just like `drafts/`. Prose scratch (candidate-pool dumps, working DocxXML, section fragments) belongs in `drafts/`; this directory is only for extracted assets. Contents are gitignored; this README is tracked to keep the directory present on clone.
