# papers/

Downloaded paper markdown — the reading agent's local paper bank.

Organized into free-form, evolving research-area folders (kebab-case, e.g. `vla/`, `world-models/`, `spatial-intelligence/`, `control/`). Each paper is filed in exactly one area. Files lead with a human-readable slug from the paper title and keep the arXiv/HF id as a suffix: `papers/<area>/<title-slug>-<id>.md` — e.g. `papers/vla/robosemanticbench-2606.02277.md`.

This is a living bank: it grows as runs accumulate and is kept to a working size over time. Contents are regenerable and gitignored; this README is tracked to document the layout and keep the directory present on clone.
